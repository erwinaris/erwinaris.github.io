in linux mint ctrl+shift+alt+DownArrow shortcuts are mapped to moving windows to different wordspaces.

shortcut ctrl+shift+alt+DownArrow and ctrl+shift+alt+UpArrow doesn’t work in linux mint

  1. open pallete ctrl+shit+P search preferrence: Open Keyboard Shortcut(JSON)

  2. add code json

{ “key”: “ctrl+shift+alt+8”, “command”: “editor.action.copyLinesUpAction”, “when”: “editorTextFocus && !editorReadonly” }, { “key”: “ctrl+shift+alt+2”, “command”: “editor.action.copyLinesDownAction”, “when”: “editorTextFocus && !editorReadonly” }
